# Calculadora de IMC
Projeto desenvolvido para praticar HTML5, CSS3 e JavaScript através da criação de uma calculadora de Índice de Massa Corporal (IMC).

Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Funcionalidades
Inserção de peso e altura
Cálculo automático do IMC
Exibição do resultado
Classificação do IMC de acordo com os valores de referência
Interface simples e responsiva
Objetivo

Aprimorar conhecimentos em JavaScript, manipulação do DOM, eventos e lógica de programação através da criação de uma aplicação funcional.

Aprendizados
Manipulação de elementos com JavaScript
Captura e validação de dados do usuário
Estruturas condicionais
Eventos de formulário
Desenvolvimento Front-End
